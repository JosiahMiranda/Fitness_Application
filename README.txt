# Fitness Website Application

This website application hosts different pages for health-related information, as well as providing the following capabilities:
- Recording Time done spent practicing Yoga
- Recording Time done spent practicing Meditation
- Recording Time done spent practicing Stretching
- A meal calculator that takes in a number of desired calory intake every day and provides a meal plan to the user.


Other functions include:
- An admin portal to show statistics of users regarding their recorded times and meal plans.
- Graphical statistic implementations.
- AJAX Search functionality.
